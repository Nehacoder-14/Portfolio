## ✨ About

This is my personal developer portfolio — a fully responsive, animated website showcasing my skills, projects, and experience as a **Full-Stack Developer** and **CS Undergraduate**.

Built with **pure HTML/CSS/JS** and **Three.js**, it features a 3D particle background, glassmorphic UI, custom cursor, tilt effects, dark/light mode, and live project demos.

🔗 **Live at:** [https://nehacoder-14.github.io/Portfolio/](https://nehacoder-14.github.io/Portfolio/)

---

## 🚀 Features

- 🎨 **3D animated background** – particles & wireframe shapes (Three.js)
- ✨ **Glassmorphism** – frosted glass panels with backdrop blur
- 🖱️ **Custom cursor** with smooth trailing effect
- 🌗 **Dark / Light mode** toggle
- 📱 **Fully responsive** – works on mobile, tablet, desktop
- 🃏 **Tilt effect** on project cards (3D hover)
- 🔍 **Scroll‑reveal animations**
- 💼 **Projects section** with live demos & GitHub links
- 📜 **Certifications & Education** sections
- 📬 **Contact form** (frontend demo)
